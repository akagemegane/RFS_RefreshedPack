# Refreshed Pack
Refreshed Pack is a multiplayer compilation addon for Blood: Refreshed Supply

Suited as a band-aid solution for playing custom maps since Refreshed Supply lacks map downloading. If anyone joins and they are stuck in the lobby menu, you don't have the map. So this might be for you.

Simply download and extract the folder to the "addons" directory of your Refreshed Supply install and there you go! Just make sure to select the addon first before joining.
